## 🎯 Fundamentación técnica

Este proyecto surge como práctica integradora para aplicar conocimientos de desarrollo web, organización colaborativa y documentación técnica. Se eligió un enfoque modular y visual para facilitar la comprensión, el mantenimiento y la escalabilidad del sistema.

- Se implementaron flujos de trabajo ágiles con ramas por tarea (`USXX-TKYY`) y tableros Kanban.
- Se documentaron errores comunes en la configuración de puertos y servicios (XAMPP, Apache, Tomcat, MySQL).
- Se desarrollaron guías visuales reutilizables para facilitar la incorporación de nuevos integrantes y fortalecer la transferencia de conocimiento.
- Se priorizó la claridad en la comunicación técnica, adaptando recursos para presentaciones, videos y wiki colaborativa.
